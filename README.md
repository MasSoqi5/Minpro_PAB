# Seafood Order App

## Deskripsi Aplikasi

Seafood Order App adalah aplikasi 

A new Flutter project.

mobile berbasis Flutter yang digunakan untuk mencatat makanan/minuman yang dipesan pelanggan Restorant seafood. Aplikasi ini memungkinkan pengguna untuk menambahkan, melihat, mengubah, dan menghapus data pesanan restoran seafood.

Aplikasi ini dibuat untuk memenuhi tugas Mini Project 1 pada mata kuliah Pemograman Aplikasi Bergerak.

## Fitur Aplikasi

1️⃣ CREATE (Tambah Data)

User bisa menambahkan pesanan baru melalui form.

2️⃣ READ (Tampilkan Data)

Semua pesanan ditampilkan dalam bentuk list.

3️⃣ DELETE (Hapus Data)

User bisa menghapus pesanan yang sudah dibuat.

4️⃣ Minimal 3 TextField

Dalam 1 form input akan ada:

Nama Pelanggan

Nama Menu Seafood

Jumlah Pesanan
