# Seafood Order App

## Deskripsi Aplikasi

Seafood Order App adalah aplikasi mobile berbasis Flutter yang digunakan untuk mencatat pemesanan yang dilakukan di restoran makanan Laut. Aplikasi ini memungkinkan pengguna untuk menambahkan, melihat, mengubah, dan menghapus data pesanan makanan/minuman.
Aplikasi ini dibuat untuk memenuhi tugas Mini Project 1 pada mata kuliah Pemograman Aplikasi Bergerak.

## Widget yang digunakan

- MaterialApp

- Scaffold

- AppBar

- TextField

- ElevatedButton

- ListView

- ListTile

- Navigator

- FloatingActionButton

- setState()

## Fitur Aplikasi

1️⃣ CREATE (Tambah Data)

User bisa menambahkan pesanan baru melalui form.

2️⃣ READ (Tampilkan Data)

Semua pesanan ditampilkan dalam bentuk list.

3️⃣ DELETE (Hapus Data)

User bisa menghapus pesanan yang sudah dibuat.

4️⃣ Minimal 3 TextField

Dalam 1 form input akan ada:

Nama Pelanggan

Nama Menu Seafood

Jumlah Pesanan
